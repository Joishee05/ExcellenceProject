Conditionals

Lesson 4 of 15

Conditional statements allow your code to make decisions and execute different code based on conditions. They're essential for creating interactive programs that respond to user input and changing data.

The If Statement

The simplest conditional - executes code only if a condition is true:

let age = 20;

if (age >= 18) {
    console.log("You are an adult.");
}
// Output: "You are an adult."

let temperature = 15;

if (temperature < 0) {
    console.log("It's freezing!");
}
// No output (condition is false)

If...Else Statement

Execute one block if true, a different block if false:

let score = 75;

if (score >= 80) {
    console.log("Grade: A");
} else {
    console.log("Grade: B or lower");
}
// Output: "Grade: B or lower"

let time = 14;

if (time < 12) {
    console.log("Good morning!");
} else {
    console.log("Good afternoon!");
}
// Output: "Good afternoon!"

If...Else If...Else Statement

Test multiple conditions in order. JavaScript stops at the first true condition:

let score = 75;

if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else if (score >= 70) {
    console.log("Grade: C");
} else if (score >= 60) {
    console.log("Grade: D");
} else {
    console.log("Grade: F");
}
// Output: "Grade: C"

How This Works

let time = 14;  // 2 PM in 24-hour format

if (time < 12) {
    console.log("Morning");      // Not executed
} else if (time < 17) {
    console.log("Afternoon");    // This runs (14 < 17 is true)
} else if (time < 21) {
    console.log("Evening");      // Skipped (found a match already)
} else {
    console.log("Night");        // Skipped
}

The Switch Statement

For comparing one value against many options. Cleaner than many if...else if blocks:

let day = 3;  // 1 = Monday, 2 = Tuesday, etc.
let dayName;

switch (day) {
    case 1:
        dayName = "Monday";
        break;
    case 2:
        dayName = "Tuesday";
        break;
    case 3:
        dayName = "Wednesday";
        break;
    case 4:
        dayName = "Thursday";
        break;
    case 5:
        dayName = "Friday";
        break;
    default:
        dayName = "Weekend";
}

console.log(dayName);  // "Wednesday"
Important: Don't Forget break;

Without break, JavaScript continues executing all cases below the match ("fall-through"). This is usually a bug:

let fruit = "apple";

switch (fruit) {
    case "apple":
        console.log("Red fruit");  // Prints this
    case "banana":
        console.log("Yellow fruit"); // And this (BUG!)
    case "grape":
        console.log("Purple fruit"); // And this (BUG!)
        break;
}

Ternary Operator (Conditional Operator)

A short way to write simple if...else statements on one line:

let age = 20;

// Traditional if...else
let status;
if (age >= 18) {
    status = "adult";
} else {
    status = "minor";
}

// Using ternary operator (much shorter)
let status2 = age >= 18 ? "adult" : "minor";

console.log(status2); // "adult"

Ternary Operator Syntax

condition ? valueIfTrue : valueIfFalse

// Example 1
let canVote = age >= 18 ? "Yes" : "No";

// Example 2
let message = score >= 60 ? "Passed" : "Failed";

// Example 3 - nested ternary
let grade = score >= 90 ? "A" : score >= 80 ? "B" : score >= 70 ? "C" : "F";

Combining Conditions

Use logical operators to test multiple conditions:

let age = 25;
let hasLicense = true;

// AND (&&) - both must be true
if (age >= 18 && hasLicense) {
    console.log("You can rent a car");
}

// OR (||) - at least one must be true
if (age < 18 || hasLicense === false) {
    console.log("You cannot rent a car");
}

// NOT (!) - inverts the condition
if (!hasLicense) {
    console.log("You need a license");
}

// Multiple AND conditions
if (age >= 18 && hasLicense && age < 70) {
    console.log("You can rent a car");
}

Conditional Flowchart

User action or data
        ↓
    [Is condition true?]
        ↙        ↖
      YES        NO
       ↓          ↓
   Run this    Check next
   code block  condition?
       ↓          ↓
   Continue   [Another condition?]
              ↙        ↖
            YES        NO
             ↓          ↓
         Run else    Continue
         code block  program

Real-World Example

let userInput = "admin";
let password = "12345";

if (userInput === "admin" && password === "12345") {
    console.log("Access granted: Admin panel");
} else if (userInput === "user" && password === "pass") {
    console.log("Access granted: User dashboard");
} else {
    console.log("Access denied: Invalid credentials");
}

Key Takeaways

  • Use if to test ONE condition
  • Use if...else for ONE choice between two paths
  • Use if...else if...else for MULTIPLE conditions
  • Use switch when comparing ONE value to MANY options
  • Use ternary operator for quick true/false decisions
  • Always use === for comparisons, not ==
