JavaScript Operators

Operators are symbols that perform operations on values and variables. JavaScript has several types of operators.

Arithmetic Operators

Used to perform mathematical calculations:

let a = 10;
let b = 3;

console.log(a + b);  // Addition: 13
console.log(a - b);  // Subtraction: 7
console.log(a * b);  // Multiplication: 30
console.log(a / b);  // Division: 3.333...
console.log(a % b);  // Modulus (remainder): 1
console.log(a ** b); // Exponentiation: 1000 (10^3)

Increment and Decrement

let count = 5;

count++;  // Increment by 1 (count = 6)
console.log(count); // 6

count--;  // Decrement by 1 (count = 5)
console.log(count); // 5

Assignment Operators

Used to assign values to variables:

let x = 10;

x += 5;  // Same as: x = x + 5  (x becomes 15)
x -= 3;  // Same as: x = x - 3  (x becomes 12)
x *= 2;  // Same as: x = x * 2  (x becomes 24)
x /= 4;  // Same as: x = x / 4  (x becomes 6)
x %= 4;  // Same as: x = x % 4  (x becomes 2)

console.log(x); // 2

Comparison Operators

Used to compare values and return true or false:

let age = 20;

console.log(age == 20);   // Equal to: true
console.log(age === 20);  // Strict equal to: true
console.log(age != 18);   // Not equal to: true
console.log(age !== "20"); // Strict not equal: true
console.log(age > 18);    // Greater than: true
console.log(age < 25);    // Less than: true
console.log(age >= 20);   // Greater than or equal: true
console.log(age <= 19);   // Less than or equal: false

Important: Use === (strict equality) instead of ==. Strict equality checks both value and type, while == performs type conversion.

console.log(5 == "5");   // true (converts string to number)
console.log(5 === "5");  // false (different types)

Logical Operators

Used to combine or invert boolean values:

let age = 20;
let hasID = true;

// AND (&&) - both must be true
console.log(age >= 18 && hasID);  // true

// OR (||) - at least one must be true
console.log(age >= 21 || hasID);  // true

// NOT (!) - inverts the value
console.log(!hasID);  // false
console.log(!(age < 18)); // true

Logical Operator Examples

let isStudent = true;
let hasDiscount = false;

// Check if eligible for student discount
let eligible = isStudent && hasDiscount;
console.log(eligible); // false

// Check if any discount applies
let anyDiscount = isStudent || hasDiscount;
console.log(anyDiscount); // true

String Operators

The + operator can also concatenate (join) strings:

let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName;
console.log(fullName); // "John Doe"

// You can also use += with strings
let message = "Hello";
message += " World";
console.log(message); // "Hello World"
