Creating an HTTP Server

Node.js includes the http module for building servers. A server listens for incoming requests and sends responses back to clients.

Request-Response Cycle

Every HTTP interaction follows the request-response pattern. The client sends a request, the server processes it, and returns a response with status code and data.

Your First Server

Use the http module to create a simple server. Listen on a port (commonly 3000 or 8080). Handle requests and send responses.

const http = require('http');
const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello World!');
});
server.listen(3000);
