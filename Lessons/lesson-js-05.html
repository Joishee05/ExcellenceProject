JavaScript Functions

Functions are reusable blocks of code that perform a specific task. They help organize your code and avoid repetition.

Function Declaration

The traditional way to create a function:

function greet() {
    console.log("Hello, World!");
}

// Call the function
greet(); // Outputs: "Hello, World!"

Functions with Parameters

Parameters allow you to pass data into functions:

function greetUser(name) {
    console.log("Hello, " + name + "!");
}

greetUser("Alice");  // "Hello, Alice!"
greetUser("Bob");    // "Hello, Bob!"

// Multiple parameters
function add(a, b) {
    console.log(a + b);
}

add(5, 3);  // 8
add(10, 20); // 30

Return Values

Functions can return values using the return keyword:

function multiply(a, b) {
    return a * b;
}

let result = multiply(4, 5);
console.log(result); // 20

// You can use the returned value directly
console.log(multiply(3, 7)); // 21

Return Statement Details

function calculateGrade(score) {
    if (score >= 90) {
        return "A";
    } else if (score >= 80) {
        return "B";
    } else if (score >= 70) {
        return "C";
    } else {
        return "F";
    }
}

console.log(calculateGrade(95)); // "A"
console.log(calculateGrade(82)); // "B"

Note: When a return statement is executed, the function stops immediately and returns the value. Any code after return won't run.

Function Expressions

You can also store functions in variables:

const square = function(x) {
    return x * x;
};

console.log(square(5)); // 25
console.log(square(8)); // 64

Arrow Functions

A shorter syntax for writing functions (ES6+):

// Traditional function
const add = function(a, b) {
    return a + b;
};

// Arrow function
const addArrow = (a, b) => {
    return a + b;
};

// Shorter arrow function (implicit return)
const addShort = (a, b) => a + b;

console.log(addShort(10, 5)); // 15

Arrow Function Examples

// Single parameter (no parentheses needed)
const double = x => x * 2;
console.log(double(4)); // 8

// No parameters (empty parentheses required)
const sayHi = () => console.log("Hi!");
sayHi(); // "Hi!"

// Multiple lines (braces and return required)
const calculateArea = (width, height) => {
    let area = width * height;
    return area;
};
console.log(calculateArea(5, 10)); // 50

Default Parameters

You can set default values for parameters:

function greet(name = "Guest") {
    console.log("Hello, " + name + "!");
}

greet("Alice");  // "Hello, Alice!"
greet();         // "Hello, Guest!"

// Arrow function with defaults
const power = (base, exponent = 2) => base ** exponent;
console.log(power(5));     // 25 (5^2)
console.log(power(5, 3));  // 125 (5^3)
