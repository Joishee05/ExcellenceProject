Meta Tags & SEO

Lesson 13 of 15

Meta tags provide information about your webpage to browsers and search engines. They live in the <head> section and don't display on the page. Meta tags help search engines understand your content and improve how your page appears in search results (SEO).

What Are Meta Tags?

Meta tags are HTML tags that give metadata (data about data) about your page. Search engines like Google use meta tags to understand your content, index pages properly, and display previews in search results. They're essential for SEO (Search Engine Optimization).

Meta Tag Structure:

<meta name="attribute" content="value">

Essential Meta Tags

1. Charset Declaration

<meta charset="UTF-8">

Defines character encoding. Always use UTF-8 for universal character support.

2. Viewport Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Makes pages responsive and mobile-friendly. Tells browsers how to scale content on different devices.

3. Description

<meta name="description" content="Brief description of your page (155-160 characters)">

Summary shown in search results. Keep it under 160 characters. This is crucial for CTR (click-through rate).

4. Keywords

<meta name="keywords" content="keyword1, keyword2, keyword3">

Search terms relevant to your page. While less important now, still useful for clarity. Limit to 5-10 keywords.

5. Author

<meta name="author" content="Your Name">

Identifies the page creator. Optional but helpful for documentation.

Open Graph Meta Tags (Social Media)

Control how your page appears when shared on social media:

<meta property="og:title" content="Page Title">
<meta property="og:description" content="Page description">
<meta property="og:image" content="https://example.com/image.jpg">
<meta property="og:url" content="https://example.com/page">
<meta property="og:type" content="website">

Twitter Card Meta Tags

Control how your page appears when shared on Twitter:

<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Page Title">
<meta name="twitter:description" content="Description">
<meta name="twitter:image" content="https://example.com/image.jpg">

Complete SEO Meta Tags Example

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Learn HTML, CSS, and JavaScript with our comprehensive tutorials">
  <meta name="keywords" content="HTML, CSS, JavaScript, web development">
  <meta name="author" content="SLU LearnCode">
  <title>Learn Web Development | SLU LearnCode</title>
</head>

SEO Best Practices

✓ DO
  • ✓ Write unique descriptions (155-160 char)
  • ✓ Use relevant keywords naturally
  • ✓ Include viewport meta tag
  • ✓ Set charset to UTF-8
  • ✓ Use descriptive page titles
✗ DON'T
  • ✗ Stuff keywords excessively
  • ✗ Copy descriptions from others
  • ✗ Forget the viewport tag
  • ✗ Use generic descriptions
  • ✗ Skip meta tags entirely

How Search Engines Use Meta Tags

Title: Shown in search results and browser tabs

Description: Snippet under title in search results

Keywords: Help categorize content (less important now)

Viewport: Affects mobile search ranking

The Title Tag (Not Technically Meta, But Critical)

<title>Best Dog Training | Expert Tips - SLU LearnCode</title>

Title Best Practices: Keep under 60 characters, include primary keyword, make it descriptive and compelling, avoid keyword stuffing.

Meta Tag Example Breakdown

Description Meta Tag:

<meta name="description" content="Learn HTML basics with interactive examples and exercises. Perfect for beginners.">
  • Name: "description" tells search engines this is a page description
  • Content: The actual description shown in search results
  • Length: 155-160 characters is ideal (Google shows 155-160)

Why Meta Tags Matter

Meta tags are essential because: They improve search engine ranking, increase click-through rates from search results, control how content appears on social media, ensure proper page rendering across devices, and help browsers understand your content. Good SEO starts with proper meta tags.
Key Takeaways
  • Meta tags go in the <head> section
  • Use <meta charset="UTF-8"> for character encoding
  • Viewport tag: <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Description: 155-160 characters, appears in search results
  • Keywords: 5-10 relevant terms (less critical for ranking)
  • Open Graph tags control social media preview
  • Title tag: Critical for SEO and browser display
  • Every page should have unique title and description
