Lesson 9: Transitions & Animations

Lesson 9 of 15

What You'll Learn

In this lesson, you will master CSS transitions and animations to create smooth, engaging visual effects. You will learn how to animate property changes, use timing functions, and create complex keyframe animations.

CSS Transitions

Transitions smoothly animate property changes from one value to another over a specified duration.

Transition Properties

transition-property: background-color; /* Which property to animate (change smoothly) */ transition-duration: 0.5s; /* How long in seconds the animation takes */ transition-timing-function: ease; /* How the animation moves (slow, fast, linear) */ transition-delay: 0.2s; /* Wait this long before starting the animation */ /* Shorthand - all properties in one line */ transition: background-color 0.5s ease 0.2s;

What Each Part Does:

  • transition-property: Tells CSS which property to smoothly animate. Change from blue to red? Use background-color. Change size? Use width or height
  • transition-duration: 0.5s: The animation takes 0.5 seconds (half a second). Longer times = slower, smoother animations
  • transition-timing-function: ease: Starts slow, speeds up in middle, slows at end (feels natural). "linear" is constant speed
  • transition-delay: 0.2s: Wait 0.2 seconds BEFORE starting. Good for animations that should happen one after another

Real Example: When you hover over a button and it smoothly changes color instead of switching instantly - that's a transition!

Transition Property

Specify which CSS properties animate.

transition-property: color; /* Single property */ transition-property: color, background-color; /* Multiple */ transition-property: all; /* All animatable properties */

Animatable Properties

Not all properties can be animated. Animatable properties include: color, background-color, width, height, padding, margin, transform, opacity, border, box-shadow, and many others. Non-animatable properties like display or font-family change instantly.

Transition Duration

The transition-duration specifies how long the animation takes.

transition-duration: 0.3s; /* 300 milliseconds */ transition-duration: 1s; /* 1 second */ transition-duration: 200ms; /* 200 milliseconds */

Transition Timing Functions

Timing functions control the acceleration and deceleration of animations.

Function Description
linear Constant speed throughout
ease Slow start, fast middle, slow end (default)
ease-in Slow start, fast end
ease-out Fast start, slow end
ease-in-out Slow start and end, fast middle

Cubic Bezier Curves

For precise timing control, use cubic-bezier function.

transition-timing-function: cubic-bezier(0.25, 0.1, 0.25, 1);

Practical Transition Examples

Button Hover Effect

button { background-color: #10b981; color: white; padding: 10px 20px; border: none; border-radius: 5px; transition: background-color 0.3s ease; cursor: pointer; } button:hover { background-color: #059669; }

Smooth Background Color Change

.box { width: 200px; height: 200px; background-color: blue; transition: background-color 0.5s ease; } .box:hover { background-color: red; }

Multiple Property Transition

.card { padding: 20px; background: white; border: 2px solid #ddd; box-shadow: 0 2px 5px rgba(0,0,0,0.1); transition: all 0.3s ease; } .card:hover { border-color: #10b981; box-shadow: 0 8px 15px rgba(0,0,0,0.3); transform: translateY(-5px); }

CSS Keyframe Animations

Keyframes allow more complex animations with multiple stages. Define animation with @keyframes rule.

@keyframes slidein { from { left: -300px; } to { left: 0; } } .element { animation-name: slidein; animation-duration: 1s; }

Animation Properties

Complete Animation Syntax

animation-name: slidein; /* Animation name */ animation-duration: 1s; /* How long to run */ animation-timing-function: ease; /* Timing curve */ animation-delay: 0.5s; /* Delay before start */ animation-iteration-count: 1; /* How many times (or infinite) */ animation-direction: normal; /* forward, backward, alternate, etc */ animation-fill-mode: forwards; /* Keep end state */ /* Shorthand */ animation: slidein 1s ease 0.5s infinite alternate forwards;

Keyframe Syntax

Define animation stages using percentages or from/to keywords.

/* Using percentages */ @keyframes bounce { 0% { transform: translateY(0); } 50% { transform: translateY(-20px); } 100% { transform: translateY(0); } } /* Using from/to */ @keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }

Animation Examples

Spinning Loader

@keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } .loader { width: 50px; height: 50px; border: 4px solid #f3f3f3; border-top: 4px solid #10b981; border-radius: 50%; animation: spin 1s linear infinite; }

Fade In Text

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } .fade-text { animation: fadeIn 2s ease-in; }

Pulse Effect

@keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } .pulse-element { animation: pulse 2s ease-in-out infinite; }

Animation Fill Mode

Value Behavior
none Default - revert to initial styles after animation
forwards Keep final animation state
backwards Apply first frame styles before animation starts
both Apply both forwards and backwards

Performance Tips

  • Use transform and opacity: These are GPU-accelerated and performant
  • Avoid animating: width, height, left, top (use transform instead)
  • Keep animations short: 200-500ms for UI feedback, 2-3s for longer effects
  • Reduce motion: Respect prefers-reduced-motion media query

Respecting User Preferences

@media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; } }
