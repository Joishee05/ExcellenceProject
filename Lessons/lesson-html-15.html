HTML iframes & Embedding Content

What is an iframe?

An iframe (inline frame) allows you to embed another HTML document within your current page.

Basic Iframe Syntax

<iframe src="https://www.example.com" 
        width="800" 
        height="600"
        title="Description">
</iframe>

Common Uses

  • Embedding YouTube videos
  • Displaying Google Maps
  • Showing external content
  • Embedding social media posts

Important Attributes

  • src - URL of the page to embed
  • width and height - Frame dimensions
  • title - Accessibility description
  • frameborder - Border around iframe (0 or 1)
  • allowfullscreen - Allow fullscreen mode
