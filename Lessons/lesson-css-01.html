What is CSS?

CSS stands for Cascading Style Sheets. It's used to style and layout web pages.

What Does CSS Do?

  • Controls colors, fonts, and spacing
  • Creates layouts and positioning
  • Adds animations and transitions
  • Makes websites responsive

How to Add CSS:

There are three ways to add CSS to HTML:

1. Inline CSS: <p style="color: blue;">Text</p>
2. Internal CSS: In a <style> tag in the <head>
3. External CSS: Link to a separate .css file (best practice!)

CSS Syntax:

selector {
    property: value;
    property: value;
}
