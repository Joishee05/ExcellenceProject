Text Formatting

HTML provides several tags to format text and add emphasis:

Common Formatting Tags:

  • <strong> - Bold text (important)
  • <em> - Italic text (emphasis)
  • <b> - Bold text (stylistic)
  • <i> - Italic text (stylistic)
  • <u> - Underlined text
  • <mark> - Highlighted text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text

Note: Use <strong> instead of <b> and <em> instead of <i> for better semantics and accessibility.
