Links

Lesson 5 of 15

Links are the glue that holds the web together. They allow users to navigate between pages, sections within a page, and external websites. The <a> (anchor) tag is one of the most important elements in HTML.

The Anchor Tag (<a>)

The <a> tag creates a hyperlink. The href attribute specifies the destination URL. Without the href attribute, the link won't work.

Basic Link Syntax:

<a href="https://www.example.com">Click here</a>
<a href="page2.html">Go to Page 2</a>
<a href="#section">Jump to Section</a>

Types of Links

Absolute URLs

What it is: Full URL starting with http:// or https://

<a href="https://google.com">Google</a>

Use for: External websites, leaving your site

Relative URLs

What it is: Path relative to current page location

<a href="pages/about.html">About</a>

Use for: Internal website links, same site

Anchor Links (Page Jumps)

What it is: Links to sections on the same page with #

<a href="#intro">Jump to Intro</a>

Use for: Navigation within long pages

Email Links

What it is: Links that open default email client

<a href="mailto:info@example.com">Email Us</a>

Use for: Contact links, email addresses

Understanding URLs

A URL (Uniform Resource Locator) is the address of a web page. Let's break down what each part means:

https://www.example.com/pages/about.html?id=123#top
https:// = Protocol (secure connection)
www.example.com = Domain name
/pages/about.html = Path to specific file
?id=123 = Query parameters
#top = Fragment/anchor (jump to this id)

Absolute vs Relative URLs Explained

Absolute URLs
https://www.google.com

Complete address: Works from any page, any site

Good for: External links, different websites

Relative URLs
../pages/about.html

Relative to current file: Shorter, more portable

Good for: Internal site navigation

Creating Anchor Links (Jump to Section)

Anchor links let users jump to a specific section on the same page. You need two parts: a link with #id and a target element with a matching id.

Step 1: Create the target with an ID

<h2 id="benefits">Why This Matters</h2>

Step 2: Create a link pointing to that ID

<a href="#benefits">Jump to Benefits Section</a>

Complete Example:

<!DOCTYPE html>
<html>
<body>
  <h1>Table of Contents</h1>
  <ul>
    <li><a href="#intro">Introduction</a></li>
    <li><a href="#benefits">Benefits</a></li>
    <li><a href="#conclusion">Conclusion</a></li>
  </ul>

  <h2 id="intro">Introduction</h2>
  <p>Content here...</p>

  <h2 id="benefits">Benefits</h2>
  <p>More content...</p>
</body>
</html>

Important Link Attributes

target="_blank"

Opens link in a new tab

<a href="page.html" target="_blank">Open</a>

title

Shows hover text (tooltip)

<a href="..." title="Click to learn">Link</a>

rel

Describes relationship to link

<a href="..." rel="nofollow">Link</a>

download

Makes link download a file

<a href="file.pdf" download>PDF</a>

Link Best Practices

✓ DO
  • Use descriptive link text
  • Use relative URLs for internal links
  • Add title attributes for tooltips
  • Use target="_blank" for external links
  • Test all links work correctly
✗ DON'T
  • Use "click here" as link text
  • Break links with dead URLs
  • Open internal links in new tabs
  • Use links for styling (use CSS)
  • Make links hard to recognize

Writing Good Link Text

The text inside a link should clearly describe where it goes. This is important for accessibility and SEO.

❌ Poor Link Text

Click here for more info

Visit our website here

Read this article

Not descriptive – doesn't explain destination

✓ Good Link Text

Learn more about our services

View our website

Read the full article about web design

Clear – describes what's at the destination

Real-World Link Examples

<!-- Navigation menu with relative URLs -->
<nav>
  <a href="index.html">Home</a>
  <a href="about.html">About Us</a>
  <a href="products.html">Products</a>
  <a href="contact.html">Contact</a>
</nav>

<!-- Footer with external and email links -->
<footer>
  <a href="https://twitter.com/company" target="_blank" title="Follow us on Twitter">Twitter</a>
  <a href="https://facebook.com/company" target="_blank" title="Like us on Facebook">Facebook</a>
  <a href="mailto:contact@example.com">Email Us</a>
</footer>

Why Links Are Critical

Understanding links is essential because: Links are what make the web interconnected. They enable navigation, create website structure, improve SEO through proper linking strategies, and provide accessibility for all users. Well-designed links make sites intuitive and user-friendly. This is one of the most fundamental HTML concepts you'll use on every website.
Key Takeaways
  • The <a> tag creates links with the href attribute
  • Absolute URLs start with http:// or https:// (external)
  • Relative URLs are paths relative to current file (internal)
  • Anchor links (with #) jump to sections on the same page
  • Use target="_blank" to open links in new tabs
  • Use descriptive link text, not "click here"
  • mailto: links open the default email client
  • Always test links to ensure they work
