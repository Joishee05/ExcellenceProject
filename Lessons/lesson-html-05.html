Links

Links are what make the web "web"! They allow you to navigate between pages.

The Anchor Tag:

Links are created using the <a> (anchor) tag:

<a href="URL">Link Text</a>

Link Attributes:

  • href - The URL or destination
  • target="_blank" - Opens in a new tab
  • title - Tooltip text on hover

Types of Links:

  • External: href="https://www.example.com"
  • Internal: href="about.html"
  • Email: href="mailto:email@example.com"
  • Phone: href="tel:+1234567890"
