Selectors

Selectors let you target specific HTML elements to style them.

Types of Selectors:

  • Element Selector: p { } - Targets all <p> elements
  • Class Selector: .classname { } - Targets elements with class="classname"
  • ID Selector: #idname { } - Targets element with id="idname"
  • Universal Selector: * { } - Targets all elements

Selector Examples:

/* Element */
h1 { color: blue; }

/* Class */
.highlight { background: yellow; }

/* ID */
#header { font-size: 24px; }

/* Multiple elements */
h1, h2, h3 { font-family: Arial; }
Note: Use classes for reusable styles and IDs for unique elements!
