CSS Filters

Filters apply visual effects to elements. They are performant and easy to use.

Available Filters

/* Blur effect - makes things fuzzy */ filter: blur(5px); /* 5px blur radius - higher numbers = blurrier */ /* Brightness adjustment */ filter: brightness(150%); /* 150% = 50% brighter than normal */ filter: brightness(50%); /* 50% = half brightness (darker) */ /* Contrast adjustment - makes light lighter and dark darker */ filter: contrast(150%); /* 150% = more dramatic contrast */ /* Grayscale - removes color (makes black & white) */ filter: grayscale(100%); /* 100% = completely black & white */ filter: grayscale(50%); /* 50% = halfway to black & white (muted colors) */

What Each Filter Does:

  • blur(5px): Makes the element blurry. Bigger number = more blur. Great for background images
  • brightness(150%): Makes brighter or darker. 100% is normal, below 100% is darker, above is brighter
  • contrast(150%): Increases the difference between light and dark. Makes images pop more
  • grayscale(100%): Removes all color. Perfect for hover effects - gray on normal, color on hover
  • Most Important: Filters are VISUAL EFFECTS ONLY. They don't change the actual image - just how it looks

Easy Example: Hover over an image and it changes from grayscale to color. That's smooth and professional looking!

Combining Filters

/* Multiple filters */ filter: brightness(110%) contrast(120%) saturate(130%); /* On images */ img { transition: filter 0.3s ease; } img:hover { filter: brightness(110%) saturate(120%); }

Grayscale Hover Effect

img { filter: grayscale(100%); transition: filter 0.3s ease; } img:hover { filter: grayscale(0%); }

Blend Modes

Blend modes define how an element blends with elements behind it.

Common Blend Modes

Mode Effect
normal Default - no blending
multiply Darkens by multiplying colors
screen Lightens like light projection
overlay Combines multiply and screen
darken Keeps darker pixels
lighten Keeps lighter pixels

Blend Mode Example

.image-overlay { position: relative; background-image: url('image.jpg'); } .overlay-color { position: absolute; top: 0; left: 0; width: 100%; height: 100%; background: rgba(16, 185, 129, 0.5); mix-blend-mode: multiply; }

Clip-Path

Clip-path creates complex shapes by clipping elements to a specified path.

Basic Shapes

/* Circle */ clip-path: circle(50%); /* Ellipse */ clip-path: ellipse(40% 30%); /* Polygon */ clip-path: polygon(50% 0%, 100% 25%, 100% 75%, 50% 100%, 0% 75%, 0% 25%); /* Inset */ clip-path: inset(10% 20% 30% 40%);

Clip-Path Examples

/* Circle image */ img.circle { clip-path: circle(50%); } /* Triangle */ .triangle { width: 200px; height: 200px; background: #10b981; clip-path: polygon(50% 0%, 100% 100%, 0% 100%); } /* Star shape */ .star { clip-path: polygon(50% 0%, 61% 35%, 98% 35%, 68% 57%, 79% 91%, 50% 70%, 21% 91%, 32% 57%, 2% 35%, 39% 35%); }

CSS Masks

Masks define which parts of an element are visible using an image or gradient.

Mask Image

/* Using an image as mask */ -webkit-mask-image: url('mask.png'); mask-image: url('mask.png'); /* Fade mask (gradient) */ -webkit-mask-image: linear-gradient(to right, transparent, black); mask-image: linear-gradient(to right, transparent, black);

Mask Examples

/* Fade image sides */ .fade-sides { -webkit-mask-image: linear-gradient(to right, transparent, black 20%, black 80%, transparent); mask-image: linear-gradient(to right, transparent, black 20%, black 80%, transparent); } /* Radial fade mask */ .fade-radial { -webkit-mask-image: radial-gradient(circle, black 60%, transparent); mask-image: radial-gradient(circle, black 60%, transparent); }

Transform Perspective

Create 3D effects using perspective.

/* Perspective container */ .container { perspective: 1000px; } /* 3D transform */ .card { transform: rotateY(45deg) rotateX(10deg); } /* Skew effect */ .skew { transform: skewX(20deg); } /* 3D rotation */ .rotate-3d { transform: rotateX(45deg) rotateY(45deg) rotateZ(45deg); }

3D Card Flip

.card-container { perspective: 1000px; width: 300px; height: 300px; } .card { position: relative; width: 100%; height: 100%; transform-style: preserve-3d; transition: transform 0.6s; } .card-container:hover .card { transform: rotateY(180deg); } .card-front, .card-back { position: absolute; width: 100%; height: 100%; backface-visibility: hidden; } .card-back { transform: rotateY(180deg); }

Backdrop Filter

Apply filters to the area behind an element.

/* Frosted glass effect */ .frosted { background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.2); } /* Navigation with blur */ nav { background: rgba(0, 0, 0, 0.5); backdrop-filter: blur(5px); }

Combining Advanced Effects

Creative Image Hover Effect

img { filter: grayscale(100%) brightness(80%); clip-path: circle(40%); transition: all 0.3s ease; } img:hover { filter: grayscale(0%) brightness(100%); clip-path: circle(50%); transform: scale(1.05); }

Frosted Glass Card

.glassmorphism { background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(10px); -webkit-backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.2); border-radius: 10px; padding: 20px; box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1); }
