Operators

Lesson 3 of 15

Operators are symbols that perform operations on values and variables. They're the foundation of programming logic, allowing you to manipulate data, make comparisons, and build complex expressions.

Arithmetic Operators

Used to perform mathematical calculations. All operators work with numbers:

let a = 10;
let b = 3;

console.log(a + b);  // Addition: 13
console.log(a - b);  // Subtraction: 7
console.log(a * b);  // Multiplication: 30
console.log(a / b);  // Division: 3.333...
console.log(a % b);  // Modulus (remainder): 1
console.log(a ** b); // Exponentiation: 1000 (10^3)

Increment and Decrement Operators

Shorthand for adding or subtracting 1:

let count = 5;

count++;  // Post-increment: count = 6, returns old value
count--;  // Post-decrement: count = 5, returns old value

++count;  // Pre-increment: increments first, then returns
--count;  // Pre-decrement: decrements first, then returns

console.log(count); // 5

Assignment Operators

Assign values to variables, often with a shorthand operation:

let x = 10;

x = 20;    // Basic assignment
x += 5;    // x = x + 5  (x becomes 25)
x -= 3;    // x = x - 3  (x becomes 22)
x *= 2;    // x = x * 2  (x becomes 44)
x /= 4;    // x = x / 4  (x becomes 11)
x %= 3;    // x = x % 3  (x becomes 2)

console.log(x); // 2

Comparison Operators

Compare two values and return a boolean (true or false). Used heavily in conditional statements:

let age = 20;

console.log(age == 20);   // Equal to: true
console.log(age === 20);  // Strict equal: true
console.log(age != 18);   // Not equal: true
console.log(age !== "20"); // Strict not equal: true
console.log(age > 18);    // Greater than: true
console.log(age < 25);    // Less than: true
console.log(age >= 20);   // Greater or equal: true
console.log(age <= 19);   // Less or equal: false
Important: == vs ===

Always use === (strict equality) instead of ==. The strict version checks both value AND type, while == performs automatic type conversion (which can cause bugs):

console.log(5 == "5");   // true (converts string to number)
console.log(5 === "5");  // false (different types)

Logical Operators

Combine or invert boolean values. Essential for complex conditions:

let age = 20;
let hasID = true;

// AND (&&) - both must be true
console.log(age >= 18 && hasID);  // true

// OR (||) - at least one must be true
console.log(age >= 21 || hasID);  // true

// NOT (!) - inverts the value
console.log(!hasID);  // false
console.log(!(age < 18)); // true

Practical Logical Operator Examples

let isStudent = true;
let hasStudent Discount = false;
let age = 22;

// Check if eligible for student discount
let eligible = isStudent && hasStudentDiscount;
console.log(eligible); // false

// Check if any condition applies
let canRegister = age >= 18 || isStudent;
console.log(canRegister); // true

String Operators

The + operator concatenates (combines) strings:

let firstName = "John";
let lastName = "Doe";

let fullName = firstName + " " + lastName;
console.log(fullName); // "John Doe"

// Compound assignment with strings
let message = "Hello";
message += " World";
console.log(message); // "Hello World"

Operator Precedence

JavaScript evaluates operators in a specific order. Operations with higher precedence are evaluated first:

Precedence (High to Low) Operators Example
1 (Highest) Exponentiation ** 2 ** 3 = 8
2 *, /, % 10 * 3 + 2 = 32 (not 50)
3 +, - 5 + 3 - 2 = 6
4 <, >, <=, >=, ==, === 5 > 3 = true
5 && true && false = false
6 (Lowest) || true || false = true

Using Parentheses to Override Precedence

// Without parentheses (multiplication first)
console.log(2 + 3 * 4);   // 14 (3*4 = 12, then 2+12)

// With parentheses (addition first)
console.log((2 + 3) * 4); // 20 (2+3 = 5, then 5*4)

Key Takeaways

  • Use arithmetic operators for math: +, -, *, /, %, **
  • Use comparison operators (===, !==, <, >) for conditions
  • Use logical operators (&&, ||, !) to combine conditions
  • Remember operator precedence - multiplication before addition, etc.
  • Use parentheses for clarity when combining multiple operators
