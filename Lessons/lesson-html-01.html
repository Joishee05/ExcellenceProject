What is HTML?

HTML stands for HyperText Markup Language. It's the standard language used to create web pages.

Key Concepts:

  • HTML provides the structure of web pages
  • HTML uses tags to mark up content
  • HTML files have a .html extension
  • Browsers read HTML and display it as web pages

HTML Tags:

HTML tags are keywords surrounded by angle brackets:

<tagname>content</tagname>

Most HTML tags come in pairs:

  • An opening tag: <h1>
  • A closing tag: </h1>
