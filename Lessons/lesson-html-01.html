What is HTML?

Lesson 1 of 15

HTML stands for HyperText Markup Language. It's the standard markup language used to create web pages. Every website you see on the internet is built with HTML.

What HTML Does

HTML provides the structure and content of web pages. It tells web browsers how to organize and display information. Think of HTML as the skeleton of a building – it creates the basic framework that everything else is built upon.

  • Organizes content into logical sections
  • Defines the meaning of different content types
  • Creates links between pages
  • Embeds images, audio, and video
  • Creates forms for user input
  • Provides metadata about the page

HTML Tags and Elements

HTML uses tags to mark up content. Tags are keywords surrounded by angle brackets:

<tagname>content</tagname>

Most tags come in pairs:

  • Opening tag – Example: <h1>
  • Content – The text between the tags
  • Closing tag – Example: </h1>

Example:

<h1>Welcome to Web Development</h1>
<p>This is my first web page.</p>
Key Tip: The forward slash in the closing tag tells the browser where the element ends. Never forget the closing tag!

HTML and CSS and JavaScript

HTML works together with two other technologies:

  • CSS – Handles styling and visual appearance (colors, fonts, layouts)
  • JavaScript – Adds interactivity and dynamic behavior (animations, user interactions)

HTML is the foundation. CSS and JavaScript build on top of it to create complete web experiences.

Why Learn HTML?

  • Foundation of the Web: HTML is the foundation of all websites. Without HTML, browsers wouldn't know how to display content
  • Easy to Learn: HTML is simple and beginner-friendly. You don't need advanced math or complex logic to get started
  • Gateway to Web Development: You need HTML to learn CSS and JavaScript. It's the starting point for any web developer
  • In-Demand Skill: HTML skills are valuable in the job market. Every web developer needs to know HTML
  • Creative Expression: Build and publish your own web content. Create websites for your business, portfolio, or just for fun

HTML Elements vs Tags

You'll often hear the terms "HTML tag" and "HTML element" used in web development. It's important to understand the difference:

An HTML tag is the code you write using angle brackets. An HTML element includes the tag, the content, and everything in between. For example:

<p>This is a paragraph.</p>

HTML Tags

Opening tag:

<p>

Closing tag:

</p>

HTML Element

Complete element:

<p>This is a paragraph.</p>

Includes opening tag, content, and closing tag

In this example, <p> and </p> are tags, and the entire thing including the text is an element.

Common HTML Tags You'll Learn

Throughout this course, you'll work with many different HTML tags. Here are the most common ones organized by category:

Text Content
  • <h1>-<h6> Headings
  • <p> Paragraphs
  • <strong> Bold text
  • <em> Italic text
Media
  • <img> Images
  • <video> Videos
  • <audio> Audio
  • <iframe> Embedded content
Navigation
  • <a> Links
  • <nav> Navigation
  • <menu> Menus
  • <button> Buttons
Structure
  • <div> Containers
  • <header> Headers
  • <main> Main content
  • <footer> Footers
Forms
  • <form> Forms
  • <input> Input fields
  • <select> Dropdowns
  • <textarea> Text areas
Lists
  • <ul> Unordered lists
  • <ol> Ordered lists
  • <li> List items
  • <table> Tables

Don't worry if these seem overwhelming right now. We'll explore each one in detail throughout this course.
