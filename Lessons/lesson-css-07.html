Lesson 7: CSS Grid

Lesson 7 of 15

What You'll Learn

In this comprehensive lesson, you will master CSS Grida powerful two-dimensional layout system for creating complex page layouts with rows and columns. Grid enables you to build responsive, sophisticated layouts that are difficult or impossible with Flexbox alone.

Introduction to CSS Grid

CSS Grid Layout is a two-dimensional layout system that allows you to design page layouts using rows and columns. Unlike Flexbox which is one-dimensional, Grid lets you control both horizontal and vertical positioning simultaneously, making it ideal for page-level layouts.

When to Use Grid vs Flexbox

Characteristic Flexbox Grid
Dimensions One-dimensional Two-dimensional
Best For Components, toolbars, navigation Page layouts, complex designs
Alignment Along one axis Both axes independently

Grid Container and Grid Items

Like Flexbox, Grid requires a container and items. Set display: grid on the parent to create a grid container.

.grid-container { display: grid; /* Create grid container */ grid-template-columns: 1fr 1fr 1fr; /* Define columns */ } .grid-item { /* Automatically grid items */ }

What This Does:

  • display: grid; - Creates a 2D grid layout (like a spreadsheet with rows AND columns). This is more powerful than Flexbox for complex layouts
  • grid-template-columns: 1fr 1fr 1fr; - Defines 3 columns. "1fr" means 1 fraction of available space. So 1fr 1fr 1fr means 3 equal columns
  • 1fr: Think of it like saying "each column gets 1 share of the space". If you did "1fr 2fr 1fr", the middle column would be twice as wide

Grid vs Flexbox: Flexbox is 1D (arranges things in a line). Grid is 2D (arranges things in rows AND columns). Use Grid for page layouts, Flexbox for component layouts.

Grid Template Columns

Define the column structure of your grid. You can specify fixed sizes, flexible sizes, or combinations.

/* Equal columns */ grid-template-columns: 1fr 1fr 1fr; /* 3 equal columns */ /* Fixed sizes */ grid-template-columns: 200px 300px 200px; /* Mix fixed and flexible */ grid-template-columns: 200px 1fr; /* Fixed left, flexible right */ /* Using repeat() */ grid-template-columns: repeat(3, 1fr); /* 3 equal columns */ grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Responsive */

Understanding fr (Fraction Unit)

The fr unit represents a fraction of available space. 1fr means "one part of the available space".

/* 3 equal columns - each takes 1/3 of space */ grid-template-columns: 1fr 1fr 1fr; /* First column fixed, rest equal */ grid-template-columns: 200px 1fr 1fr; /* 200px + equal remaining space */ /* Proportional */ grid-template-columns: 1fr 2fr 1fr; /* 1:2:1 ratio */

Grid Template Rows

Similarly, you can define row heights.

/* Equal rows */ grid-template-rows: 100px 200px 100px; /* Flexible rows */ grid-template-rows: auto 1fr auto; /* Auto for content, 1fr for flexible */ /* Repeat function */ grid-template-rows: repeat(3, 200px);

Gap: Spacing in Grid

The gap property creates space between grid items.

gap: 20px; /* Space between rows and columns */ column-gap: 20px; /* Space between columns only */ row-gap: 15px; /* Space between rows only */

Placing Items on the Grid

Control where grid items appear using grid-column and grid-row.

/* Make an item span 2 columns */ .item { grid-column: 1 / 3; /* Start at column 1, end at column 3 */ } /* Span syntax */ .item { grid-column: span 2; /* Span 2 columns */ } /* Row placement */ .item { grid-row: 1 / 3; /* Span from row 1 to row 3 */ }

Named Grid Areas

Use grid-template-areas for a clearer, more visual layout definition.

.grid { display: grid; grid-template-columns: 1fr 3fr 1fr; grid-template-rows: auto 1fr auto; grid-template-areas: "header header header" "sidebar main aside" "footer footer footer"; } header { grid-area: header; } .sidebar { grid-area: sidebar; } main { grid-area: main; } aside { grid-area: aside; } footer { grid-area: footer; }

Alignment in Grid

Justify-Items and Align-Items

justify-items: center; /* Center items horizontally */ align-items: center; /* Center items vertically */

Justify-Content and Align-Content

These align the entire grid within its container.

justify-content: center; /* Center grid horizontally */ align-content: center; /* Center grid vertically */

Auto-Fit and Auto-Fill

Create responsive grids that automatically adjust column count based on available space.

/* Responsive grid: auto-fit */ grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* Difference: auto-fit collapses empty tracks, auto-fill keeps them */ grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

Practical Grid Examples

Classic Page Layout

.page { display: grid; grid-template-columns: 280px 1fr; grid-template-rows: auto 1fr auto; height: 100vh; } header { grid-column: 1 / -1; } /* Span all columns */ nav { grid-column: 1; grid-row: 2; } main { grid-column: 2; grid-row: 2; } footer { grid-column: 1 / -1; } /* Span all columns */

Image Gallery with Responsive Grid

.gallery { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 15px; } .gallery-item { aspect-ratio: 1; /* Square items */ object-fit: cover; }

Grid Browser Support

CSS Grid is supported in all modern browsers. Older browsers like Internet Explorer 11 have partial or no support.
