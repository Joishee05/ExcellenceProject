Colors & Backgrounds

CSS offers many ways to add colors and backgrounds to your elements.

Color Values:

  • Named colors: color: red;
  • Hex codes: color: #FF0000;
  • RGB: color: rgb(255, 0, 0);
  • RGBA (with transparency): color: rgba(255, 0, 0, 0.5);

Background Properties:

  • background-color - Solid color
  • background-image - Image or gradient
  • background-size - Size of background
  • background-position - Position of background
