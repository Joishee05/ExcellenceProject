Images

Lesson 6 of 15

Images make websites visually engaging and convey information quickly. The <img> tag embeds images into your HTML. Unlike the <a> tag, the <img> tag is self-closing (no closing tag needed).

The Image Tag (<img>)

The <img> tag inserts an image into your webpage. It has two essential attributes that you should always include: src (the image file path) and alt (alternative text describing the image).

Basic Image Syntax:

<img src="image.jpg" alt="Description of image">
<img src="../images/photo.png" alt="A scenic mountain landscape">
<img src="https://example.com/pic.jpg" alt="Product photo">

Essential Image Attributes

src Attribute

Purpose: Specifies the path to the image file (REQUIRED)

<img src="logo.jpg" alt="...">

Can be relative path or absolute URL

alt Attribute

Purpose: Describes image for accessibility (REQUIRED)

<img src="..." alt="Product photo">

Displays if image fails to load, helps screen readers

width & height

Purpose: Sets image dimensions in pixels

<img ... width="300" height="200">

Improves page loading and prevents layout shift

title Attribute

Purpose: Shows tooltip on hover

<img ... title="Hover text">

Additional context displayed on mouseover

Why Alt Text Matters

Alt text (alternative text) is critical for web accessibility and SEO. Here's why it's so important:

  • Accessibility: Screen reader users depend on alt text to understand images
  • Broken Images: If an image fails to load, alt text displays instead
  • SEO: Search engines use alt text to understand image content
  • Performance: Users on slow connections see alt text while images load
  • Mobile Optimization: Alt text helps users who disable image loading

Writing Good Alt Text

❌ Poor Alt Text

<img src="pic.jpg" alt="image">

<img src="pic.jpg" alt="picture">

<img src="pic.jpg" alt="">

Too vague or empty – doesn't describe what's in the image

✓ Good Alt Text

<img src="..." alt="Smiling dog on grass">

<img src="..." alt="React logo">

<img src="..." alt="Mountain landscape at sunset">

Descriptive – clearly describes image content

Image Formats and File Types

Different image formats are optimized for different purposes. Choosing the right format improves website performance:

JPG / JPEG

Best for: Photos, complex images with many colors

Good compression, smaller file sizes

PNG

Best for: Graphics, logos, images with transparency

Supports transparency, lossless compression

GIF

Best for: Animations, simple graphics

Supports animations, limited colors

WebP

Best for: Modern web, smaller file sizes

Better compression, newer format

Responsive Images

Images should adapt to different screen sizes. You can make images responsive in several ways:

1. Width and Percentage

<img src="photo.jpg" alt="Photo" width="100%" style="max-width: 600px;">

2. Srcset for Different Resolutions

<img src="photo.jpg" alt="Photo"
      srcset="photo-small.jpg 480w,
                        photo-medium.jpg 800w,
                        photo-large.jpg 1200w">

Image in Context: <figure> and <figcaption>

For important images, use <figure> and <figcaption> tags to provide semantic context:

<figure>
  <img src="sunset.jpg" alt="Beautiful sunset over ocean">
  <figcaption>A sunset over the Pacific Ocean</figcaption>
</figure>

Benefits of this approach:

  • Groups image with caption semantically
  • Screen readers announce the figure element
  • Easier to style with CSS
  • Clearer content structure

Image Best Practices

✓ DO
  • Always include alt text
  • Optimize image file sizes
  • Use appropriate formats (JPG, PNG, WebP)
  • Specify width/height attributes
  • Use responsive image techniques
✗ DON'T
  • Skip alt text
  • Use giant unoptimized images
  • Use images for text (use HTML text)
  • Rely on browser to resize huge images
  • Use broken image links

Real-World Image Examples

<!-- Simple image with alt text -->
<img src="logo.png" alt="Company logo" width="200">

<!-- Product image with figure caption -->
<figure>
  <img src="product.jpg" alt="Blue coffee mug" width="300">
  <figcaption>Premium ceramic coffee mug - $24.99</figcaption>
</figure>

<!-- Responsive hero image -->
<img src="hero.jpg" alt="Mountain landscape" width="100%" style="max-width: 1200px;">

Why Images Matter

Understanding images is crucial because: Images significantly impact user engagement, page aesthetics, and website functionality. Proper image implementation with descriptive alt text ensures your site is accessible to all users, improves SEO rankings, and enhances page performance. Images are one of the most commonly used HTML elements in modern web design.
Key Takeaways
  • The <img> tag is self-closing (no closing tag)
  • src attribute is required – specifies image file path
  • alt attribute is required – describes image for accessibility
  • Alt text improves accessibility and SEO
  • Always write descriptive, meaningful alt text
  • Choose the right image format: JPG, PNG, GIF, or WebP
  • Use responsive images with srcset or CSS width
  • Use <figure> and <figcaption> for semantic image grouping
