Images

Images make your website visually appealing. Let's learn how to add them!

The Image Tag:

<img src="image.jpg" alt="Description">

Important: The <img> tag is self-closing (no closing tag needed)!

Image Attributes:

  • src - Image source (URL or file path) - Required
  • alt - Alternative text (for accessibility) - Required
  • width - Image width in pixels
  • height - Image height in pixels
  • title - Tooltip on hover

Image Sources:

  • Local: src="images/photo.jpg"
  • Online: src="https://example.com/image.jpg"
