Lesson 6: Flexbox

Lesson 6 of 15

What You'll Learn

In this comprehensive lesson, you will master Flexbox (Flexible Box Layout)one of the most important CSS layout tools. You will learn to create flexible, responsive one-dimensional layouts with powerful alignment and distribution properties.

Introduction to Flexbox

Flexbox is a CSS layout module that provides an efficient way to arrange, distribute, and align items within a container. It was introduced to solve common layout challenges and has become the standard for modern web layouts. Flexbox excels at creating responsive designs, navigation bars, forms, and components.

When to Use Flexbox

  • One-dimensional layouts: Items flowing in a single row or column
  • Navigation bars: Horizontal or vertical menus
  • Forms: Aligning labels with inputs
  • Toolbars: Grouping related controls
  • Centering content: Perfectly center items both horizontally and vertically
  • Responsive layouts: Easily adapt to different screen sizes

Flex Container and Flex Items

To use Flexbox, you create a flex container by setting display: flex on a parent element. All direct children automatically become flex items and can be controlled with flex properties.

.container { display: flex; /* Create flex container */ } .item { /* Automatically flex items */ }

What This Does:

  • display: flex; - This magic line transforms the container. All child elements inside become flex items that can be arranged flexibly
  • .item { }: Child elements automatically become flex items. You don't need to add any special CSS to them - they inherit flex behavior
  • No floats, no positioning needed: Flexbox handles all the layout for you. Just add display: flex to the parent!

Main Axis and Cross Axis

Flexbox operates on two axes:

  • Main Axis: The primary direction items flow. Default is horizontal (left to right).
  • Cross Axis: The axis perpendicular to main axis. Default is vertical (top to bottom).

Understanding these axes is critical because many properties align items along these directions.

Flex Direction: Controlling Flow

The flex-direction property determines the direction of the main axis.

flex-direction: row; /* Default: horizontal left-to-right */ flex-direction: row-reverse; /* Horizontal right-to-left */ flex-direction: column; /* Vertical top-to-bottom */ flex-direction: column-reverse; /* Vertical bottom-to-top */

Practical Direction Examples

/* Horizontal navigation */ .navbar { display: flex; flex-direction: row; /* Items flow left to right */ } /* Vertical sidebar menu */ .sidebar { display: flex; flex-direction: column; /* Items stack vertically */ }

Justify-Content: Distributing Along Main Axis

The justify-content property controls how flex items are aligned and how free space is distributed along the main axis.

justify-content: flex-start; /* Items at start (default) */ justify-content: flex-end; /* Items at end */ justify-content: center; /* Items centered */ justify-content: space-between; /* Space between items, at edges */ justify-content: space-around; /* Space around each item */ justify-content: space-evenly; /* Equal space everywhere */

Justify-Content Visual Comparison

  • flex-start: Item Item Item | | | (items at left)
  • flex-end: | | | Item Item Item (items at right)
  • center: | Item Item Item | (items in middle)
  • space-between: Item | | Item | | Item (edges at container edges)
  • space-around: | Item | Item | Item | (equal space around each)
  • space-evenly: | Item | Item | Item | (equal space everywhere)

Align-Items: Aligning Along Cross Axis

The align-items property controls how flex items are aligned along the cross axis. This is perfect for vertical centering.

align-items: stretch; /* Items stretch to fill (default) */ align-items: flex-start; /* Items at start of cross axis */ align-items: flex-end; /* Items at end of cross axis */ align-items: center; /* Items centered on cross axis */ align-items: baseline; /* Items aligned by baseline */

Perfect Centering Example

/* Center content both ways */ .hero { display: flex; justify-content: center; /* Horizontal center */ align-items: center; /* Vertical center */ height: 100vh; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); }

Gap: Spacing Between Items

The gap property creates consistent spacing between flex items. This is cleaner than using margins.

gap: 20px; /* Space between items */ row-gap: 10px; /* Space between rows */ column-gap: 20px; /* Space between columns */

Flex Property: Growing and Shrinking Items

The flex property is a shorthand for flex-grow, flex-shrink, and flex-basis. It controls how items grow and shrink.

Flex-Grow: Make Items Expand

flex-grow: 0; /* Default: do not grow */ flex-grow: 1; /* Grow equally with siblings */ flex-grow: 2; /* Grow twice as much */

Flex-Shrink: Make Items Shrink

flex-shrink: 1; /* Default: shrink if needed */ flex-shrink: 0; /* Never shrink */

Flex-Basis: Initial Size

flex-basis: auto; /* Size based on content */ flex-basis: 200px; /* Fixed initial size */ flex-basis: 50%; /* Percentage of container */

Flex Shorthand

/* flex: grow shrink basis */ flex: 1; /* flex: 1 1 0 */ flex: 0 0 200px; /* Fixed 200px, doesn't grow/shrink */ flex: 1 1 auto; /* Flexible, based on content */

Flex-Wrap: Multiple Lines

By default, all flex items fit on one line. Use flex-wrap to allow items to wrap to multiple lines.

flex-wrap: nowrap; /* Default: one line */ flex-wrap: wrap; /* Wrap to next line if needed */ flex-wrap: wrap-reverse; /* Wrap in reverse */

Responsive Card Grid

.card-container { display: flex; flex-wrap: wrap; gap: 20px; } .card { flex: 1 1 calc(33.333% - 20px); /* 3 cards per row */ min-width: 250px; /* Responsive: stack on small screens */ }

Practical Flexbox Layouts

Navigation Bar

.navbar { display: flex; justify-content: space-between; align-items: center; background: #333; padding: 15px 30px; } .logo { color: white; font-weight: bold; flex-shrink: 0; /* Don't shrink logo */ } .nav-links { display: flex; gap: 30px; flex: 1; /* Take remaining space */ }

Centered Hero Section

.hero { display: flex; justify-content: center; align-items: center; height: 100vh; background: url('background.jpg') center/cover; } .hero-content { text-align: center; color: white; }

Common Flexbox Issues and Solutions

  • Items not centering: Make sure to use both justify-content and align-items
  • Unequal item sizes: Use flex: 1 to make items equal width
  • Content overflowing: Use min-width: 0 on flex items to allow shrinking below content size
  • Last item wrapping: Use flex-wrap: wrap if items should be on multiple lines

Browser Support

Flexbox is supported in all modern browsers. For older browsers (Internet Explorer 11), some properties may need vendor prefixes.
