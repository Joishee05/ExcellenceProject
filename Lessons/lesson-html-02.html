HTML Structure

Every HTML document follows a basic structure. Let's break it down:

The Basic Structure:

<!DOCTYPE html> ‚Üê Declares this is HTML5
<html> ‚Üê Root element
  <head> ‚Üê Contains meta information
    <title></title> ‚Üê Page title (shown in browser tab)
  </head>
  <body> ‚Üê Visible page content
  </body>
</html>

Understanding Each Part:

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html> - The root element that contains all other elements
  • <head> - Contains metadata, title, links to CSS, etc.
  • <title> - Sets the page title (appears in browser tab)
  • <body> - Contains all visible content
