HTML Structure

Lesson 2 of 15

Every HTML document follows a consistent structure. This structure tells the browser how to organize and display your content. Understanding this structure is essential before you write any HTML code.

The HTML Document Structure

Every HTML document must follow a specific template. This template includes several key elements that work together to create a valid web page:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <!-- Page content goes here -->
  </body>
</html>

This basic structure is the foundation for every web page you create. Every HTML file must contain these elements in this order.

Understanding Each Part

Let's break down each part of the HTML structure and understand what it does:

<!DOCTYPE html>

This is the first line of every HTML document. It declares that this is an HTML5 document. The browser uses this declaration to understand how to interpret the rest of your code. Without it, browsers might display your page incorrectly.

<html> Element

This is the root element that contains all other elements in your document. Everything else belongs inside the opening and closing <html> tags. This element tells the browser where your HTML begins and ends.

<head> Element

The head section contains information about your page, but this information is not directly displayed on the page. The head section includes:

  • The page title (shown in the browser tab)
  • Links to stylesheets (CSS files)
  • Links to scripts (JavaScript files)
  • Character encoding information
  • Viewport settings for responsive design
  • Meta information for search engines

<title> Element

The title appears in the browser tab and tells both users and search engines what your page is about. Always give your pages descriptive titles. A good title is important for search engine optimization (SEO).

<title>My First Website - Welcome!</title>

<body> Element

Everything inside the body tags is what visitors see on your page. This includes all the text, images, links, and other content. All visible content must be placed inside the body element.

The Complete HTML Template

Here's a more complete HTML template with additional important elements:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website Title</title>
  </head>
  <body>
    <h1>Welcome to My Website!</h1>
    <p>This is my first web page.</p>
  </body>
</html>

Key Elements Explained

The complete template includes two important meta tags in the head section:

Character Encoding

<meta charset="UTF-8">

Tells the browser to use UTF-8 encoding, which supports all languages and special characters. Always include this in every HTML document.

Responsive Viewport

<meta name="viewport"...>

Makes your page responsive on mobile devices. This ensures your page displays correctly on all screen sizes.

Why Structure Matters

Proper HTML structure is important for several reasons:

  • Browser Compatibility: Valid structure ensures your page works in all modern browsers
  • Search Engine Optimization: Search engines rely on proper structure to understand your content
  • Accessibility: Screen readers and assistive technologies use structure to help users navigate
  • Maintainability: Well-structured code is easier to read, update, and debug
  • Responsive Design: Proper structure makes it easier to make your site mobile-friendly

Creating Your First HTML File

Follow these steps to create your first HTML document:

  1. Open a text editor (Notepad, VS Code, Sublime Text, etc.)
  2. Copy the complete HTML template from above
  3. Replace the title with something meaningful
  4. Add your content in the body section
  5. Save the file with a .html extension (example: index.html)
  6. Open the file in a web browser
Important: When saving HTML files, always use the .html extension. This tells your computer that the file is an HTML document. Common filenames are index.html (for home pages), about.html, contact.html, etc.

Head vs Body

It's common for beginners to get confused about what goes in the head section and what goes in the body. Here's a quick reference:

In the <head>
  • Meta tags
  • Page title
  • Links to CSS files
  • Script imports
  • Character encoding
In the <body>
  • Headings and paragraphs
  • Images and videos
  • Links
  • Forms and inputs
  • All visible content

Common HTML File Names

When naming your HTML files, follow these conventions:

File Name
  • index.html Home page
  • about.html About page
  • contact.html Contact page
  • services.html Services page
Naming Rules
  • Use lowercase letters
  • Use hyphens instead of spaces
  • Always use .html extension
  • Use descriptive names

Your First HTML Document

Now that you understand the structure, you're ready to create your first HTML document. Here's a complete example you can use as a starting point:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Website</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first web page. I'm learning HTML!</p>
  </body>
</html>
