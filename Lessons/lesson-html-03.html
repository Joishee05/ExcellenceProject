Headings & Paragraphs

Headings and paragraphs are fundamental HTML elements for organizing text content.

Headings (h1 - h6):

HTML has 6 levels of headings, from <h1> (most important) to <h6> (least important):

<h1> - Main heading (use only once per page)
<h2> - Section heading
<h3> - Sub-section heading
<h4> - Minor heading
<h5> - Even smaller heading
<h6> - Smallest heading

Paragraphs:

The <p> tag defines a paragraph. Browsers automatically add space before and after paragraphs.
