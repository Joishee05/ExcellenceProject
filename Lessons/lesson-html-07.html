Lists

Lists help organize information clearly. HTML provides three types of lists.

1. Unordered Lists (<ul>):

For bullet-point items (order doesn't matter).

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

2. Ordered Lists (<ol>):

For numbered items (order matters).

<ol>
    <li>First step</li>
    <li>Second step</li>
</ol>

3. Description Lists (<dl>):

For terms and definitions.

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
</dl>

Nested Lists:

You can nest lists inside other lists for hierarchical information.
