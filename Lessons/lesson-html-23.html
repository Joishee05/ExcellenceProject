Building Your First HTMX App

Lesson 23 of 23 - Final Lesson

Congratulations!

You've made it through all the HTMX lessons! Now let's bring it all together and build a complete interactive application. We'll create a simple note-taking app that demonstrates everything you've learned.

Project: Simple Note App

Your app will:

  • Display a list of notes
  • Add new notes dynamically
  • Delete notes with confirmation
  • Search/filter notes in real-time
  • All without page reloads

HTML Structure

<!DOCTYPE html> <html> <head> <title>My Notes App</title> <script src="https://unpkg.com/htmx.org"></script> <style> body { font-family: Arial, sans-serif; max-width: 600px; margin: 50px auto; } .note { border: 1px solid #ddd; padding: 10px; margin: 10px 0; border-radius: 5px; } button { padding: 5px 10px; background: #3b82f6; color: white; border: none; border-radius: 4px; cursor: pointer; } input { padding: 8px; width: 100%; margin-bottom: 10px; border: 1px solid #ddd; border-radius: 4px; } </style> </head> <body> <h1>My Notes</h1> <!-- Search Input --> <input type="text" placeholder="Search notes..." hx-get="/search-notes" hx-trigger="keyup delay:300ms" hx-target="#notes-list" name="query"> <!-- Add Note Form --> <form hx-post="/add-note" hx-target="#notes-list" hx-swap="beforeend"> <input type="text" name="note" placeholder="Add a new note..."> <button>Add Note</button> </form> <!-- Notes List --> <div id="notes-list"> <div class="note">Example: Learn HTMX</div> <div class="note">Example: Build a project</div> </div> </body> </html>

Server Endpoints (Python Flask Example)

from flask import Flask, request app = Flask(__name__) notes = ["Learn HTMX", "Build a project"] @app.route('/add-note', methods=['POST']) def add_note(): note = request.form.get('note') if note: notes.append(note) return render_note(note) @app.route('/search-notes') def search(): query = request.args.get('query', '') filtered = [n for n in notes if query.lower() in n.lower()] return ''.join(render_note(n) for n in filtered) def render_note(note): return f'<div class="note">{note} <button hx-delete="/delete-note" hx-confirm="Delete?">Delete</button></div>'

Key Learning Points Used

  • hx-post - Add notes without page reload
  • hx-get - Search notes in real-time
  • hx-delete - Remove notes
  • hx-target="#notes-list" - Where results go
  • hx-swap="beforeend" - Add to bottom of list
  • hx-trigger="keyup delay:300ms" - Search as you type
  • hx-confirm - Confirmation dialog before delete

What's Next?

Now that you understand HTMX fundamentals, you can:

  • Build more complex interactive applications
  • Integrate HTMX with your backend (Python, Node, Ruby, etc.)
  • Explore advanced HTMX features like websockets and polling
  • Use HTMX alongside other libraries and frameworks
  • Create real-time collaborative apps

Best Practices to Remember

  • Keep responses lightweight (just return the HTML you need)
  • Use semantic HTML for better accessibility
  • Add debounce/throttle to frequent triggers (search, scroll)
  • Always include error handling on the server
  • Use hx-confirm for destructive actions
  • Test with JavaScript disabled to ensure progressive enhancement
