DOM Manipulation

The Document Object Model (DOM) is a programming interface for HTML documents. JavaScript can manipulate the DOM to change the content, structure, and style of web pages.

What is the DOM?

The DOM represents your HTML document as a tree of objects. Each HTML element becomes a node that JavaScript can access and modify.

// HTML:
// <div id="container">
//     <h1>Title</h1>
//     <p class="text">Paragraph</p>
// </div>

// JavaScript can access and modify all of this!

Selecting Elements

getElementById()

Select an element by its ID:

// HTML: <div id="myDiv">Hello</div>

let element = document.getElementById("myDiv");
console.log(element); // <div id="myDiv">Hello</div>

querySelector()

Select the first element that matches a CSS selector:

// Select by ID
let byId = document.querySelector("#myDiv");

// Select by class
let byClass = document.querySelector(".myClass");

// Select by tag
let byTag = document.querySelector("p");

// Complex selector
let complex = document.querySelector("div.container p.text");

querySelectorAll()

Select all elements that match a selector:

// Get all paragraphs
let allParagraphs = document.querySelectorAll("p");

// Get all elements with class "item"
let allItems = document.querySelectorAll(".item");

// Loop through results
allParagraphs.forEach(function(p) {
    console.log(p.textContent);
});

Modifying Content

textContent

Get or set the text content of an element:

let heading = document.querySelector("h1");

// Get text
console.log(heading.textContent); // "Original Title"

// Set text
heading.textContent = "New Title";

innerHTML

Get or set the HTML content (can include tags):

let div = document.querySelector("#myDiv");

// Set HTML content
div.innerHTML = "<p>New <strong>HTML</strong> content</p>";

// Get HTML content
console.log(div.innerHTML);

Security Note: Be careful with innerHTML when using user input, as it can create security vulnerabilities (XSS attacks). Use textContent for plain text.

Modifying Styles

let box = document.querySelector(".box");

// Change individual styles
box.style.backgroundColor = "blue";
box.style.color = "white";
box.style.padding = "20px";
box.style.fontSize = "18px";

// Note: CSS properties with hyphens become camelCase
// background-color → backgroundColor
// font-size → fontSize

Working with Classes

let element = document.querySelector(".item");

// Add a class
element.classList.add("active");

// Remove a class
element.classList.remove("hidden");

// Toggle a class (add if not present, remove if present)
element.classList.toggle("highlighted");

// Check if class exists
if (element.classList.contains("active")) {
    console.log("Element is active");
}

// Add multiple classes
element.classList.add("large", "bold", "blue");

Modifying Attributes

let image = document.querySelector("img");
let link = document.querySelector("a");

// Get attribute
let src = image.getAttribute("src");
console.log(src);

// Set attribute
image.setAttribute("src", "new-image.jpg");
image.setAttribute("alt", "New description");

// Direct access to common attributes
link.href = "https://slu.edu";
image.src = "photo.jpg";

// Remove attribute
image.removeAttribute("title");

Creating and Adding Elements

// Create a new element
let newParagraph = document.createElement("p");

// Add content
newParagraph.textContent = "This is a new paragraph";

// Add classes/styles
newParagraph.classList.add("new-text");
newParagraph.style.color = "blue";

// Add to the page
let container = document.querySelector("#container");
container.appendChild(newParagraph);

// Create and add in one flow
let newDiv = document.createElement("div");
newDiv.innerHTML = "<h2>New Section</h2><p>Content</p>";
document.body.appendChild(newDiv);

Removing Elements

let element = document.querySelector(".remove-me");

// Remove the element
element.remove();

// Alternative (older method)
// element.parentNode.removeChild(element);
