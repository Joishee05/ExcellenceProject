Loops in JavaScript

Loops allow you to execute code repeatedly. They're essential for iterating over data and performing repetitive tasks.

For Loop

The for loop is used when you know how many times you want to loop:

// Syntax: for (initialization; condition; increment)
for (let i = 1; i <= 5; i++) {
    console.log("Count: " + i);
}
// Output:
// Count: 1
// Count: 2
// Count: 3
// Count: 4
// Count: 5

For Loop Breakdown

  • Initialization: let i = 1 - runs once at the start
  • Condition: i <= 5 - checked before each iteration
  • Increment: i++ - runs after each iteration

More For Loop Examples

// Count backwards
for (let i = 5; i >= 1; i--) {
    console.log(i);
}

// Count by 2s
for (let i = 0; i <= 10; i += 2) {
    console.log(i); // 0, 2, 4, 6, 8, 10
}

// Loop through array
let fruits = ["apple", "banana", "orange"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

While Loop

The while loop continues as long as a condition is true:

let count = 1;

while (count <= 5) {
    console.log("Count: " + count);
    count++;
}
// Same output as the for loop above

While Loop Example

let password = "";
let attempts = 0;

while (password !== "secret" && attempts < 3) {
    // In real code, you'd get input from user
    password = "attempt" + attempts;
    attempts++;
    console.log("Attempt " + attempts);
}

if (password === "secret") {
    console.log("Access granted!");
} else {
    console.log("Access denied!");
}

Do...While Loop

Similar to while, but guarantees at least one execution:

let num = 1;

do {
    console.log("Number: " + num);
    num++;
} while (num <= 5);

// The do...while loop always runs at least once,
// even if the condition is false from the start:
let x = 10;
do {
    console.log("This runs once");
} while (x < 5); // Condition is false, but code ran once

Break Statement

The break statement exits a loop early:

for (let i = 1; i <= 10; i++) {
    if (i === 5) {
        break; // Stop the loop when i equals 5
    }
    console.log(i);
}
// Output: 1, 2, 3, 4

// Finding a value
let numbers = [10, 20, 30, 40, 50];
let target = 30;

for (let i = 0; i < numbers.length; i++) {
    if (numbers[i] === target) {
        console.log("Found " + target + " at index " + i);
        break; // Stop searching once found
    }
}

Continue Statement

The continue statement skips the current iteration:

// Skip even numbers
for (let i = 1; i <= 5; i++) {
    if (i % 2 === 0) {
        continue; // Skip even numbers
    }
    console.log(i);
}
// Output: 1, 3, 5

// Skip negative numbers
let numbers = [5, -2, 8, -1, 12];

for (let i = 0; i < numbers.length; i++) {
    if (numbers[i] < 0) {
        continue; // Skip negative numbers
    }
    console.log(numbers[i]);
}
// Output: 5, 8, 12

Nested Loops

You can put loops inside other loops:

// Multiplication table
for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log(i + " x " + j + " = " + (i * j));
    }
}

// Creating a pattern
for (let i = 1; i <= 3; i++) {
    let row = "";
    for (let j = 1; j <= i; j++) {
        row += "* ";
    }
    console.log(row);
}
// Output:
// *
// * *
// * * *

Tip: Be careful with loops! Make sure they have a way to end, or you'll create an infinite loop that can freeze your browser.
