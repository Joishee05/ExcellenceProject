DigitalBonesBox Project

Learn by exploring a real-world web application built with HTML, CSS, and JavaScript

What is DigitalBonesBox?

DigitalBonesBox is an interactive web application for studying human anatomy. Instead of static textbooks or PowerPoint slides, users can:

  • Search for specific bones by name
  • View detailed anatomical images
  • Read descriptions and learn about bone functions
  • Filter and organize information

Your Task: Learn how each file works together to create this application. Each file you explore teaches you real programming techniques used in production websites.

How It All Works Together

The Architecture Flow

Step 1: User Opens App

boneset.html loads in the browser with HTML structure and CSS styling

Step 2: JavaScript Starts

main.js, api.js, and other JS files load and initialize the application

Step 3: Request Data

api.js contacts server.js to fetch bone data from the database

Step 4: Display Data

The data is formatted with style.css and displayed on the page

Step 5: User Interacts

When users search, filter, or click, JavaScript files respond instantly

Step 6: Real-Time Updates

New data is fetched and displayed without refreshing the page
