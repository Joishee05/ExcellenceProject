How to Use These Components

All these data visualization components are now available in your tutorial.css stylesheet. You can add them to any lesson page using the following HTML structure:

Example: Stats Cards

<div class="stats-container">
    <div class="stat-card">
        <div class="stat-number">37</div>
        <div class="stat-label">Total Lessons</div>
    </div>
</div>

Example: Progress Bar

<div class="progress-visual">
    <div class="progress-visual-header">
        <div class="progress-visual-title">Overall Completion</div>
        <div class="progress-visual-percentage">32%</div>
    </div>
    <div class="progress-bar-track">
        <div class="progress-bar-fill" style="width: 32%"></div>
    </div>
</div>
