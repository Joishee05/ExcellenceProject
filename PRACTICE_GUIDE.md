## Common Challenges

### HTML Lessons
- Use opening and closing tags: `<tag>content</tag>`
- Don't forget angle brackets: `<h1>` not `h1`
- Indent your code for readability

### CSS Lessons
- Property format: `property: value;` (with colon and semicolon)
- Color values: names, hex (#FFFFFF), rgb(255,0,0)
- Font sizes: pixels (36px), ems (2em), or percentages (120%)

### JavaScript Lessons
- Semicolons at end of statements (mostly)
- Use `console.log()` to see output
- Variables: `let name = "value";`
- Remember: JavaScript is case-sensitive!
